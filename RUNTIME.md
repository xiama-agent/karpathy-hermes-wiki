# 龙虾运行时指令 (v3.0)

1. 读 IRON_RULES.md → 检查 7 条铁律
2. 读 index.md → 获取知识指针
3. 回复带 topic-detection → {"wiki_ids": ["xxx"]}
4. 路径问题先读 paths.md → 不凭印象
5. 召回成功更新 trust → node update-trust.js
6. 工具失败记 RSI → node rsi-log.js
7. 写入后更新 INDEX → 追加链接
8. 对话结束写日志 → node session-summary.js
